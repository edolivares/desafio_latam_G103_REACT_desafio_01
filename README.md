## Notas Técnicas

Los endpoints del backend fueron modificados para que los productos carguen con slug e id. El formato de ID utilizado en el proyecto es numérico y no alfanumérico como estaba originalmente en el backend. Por lo tanto, las rutas funcionan como `/pizza/pizza-napolitana` o `/pizza/1`, pero no como `/pizza/p001` como se solicitó originalmente.
